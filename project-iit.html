Indian Institute of Technology, Kharagpur: ML Intern

Project Overview

[cite_start]During my summer internship under Professor Pabitra Mitra, I contributed to an agricultural project focused on **boosting productivity** by applying advanced machine learning techniques[cite: 17, 18]. This work involved the practical application of deep learning for **image classification** to predict plant leaf count.

Key Contributions & Technical Detail

Model Development and Accuracy

    [cite_start]
  • Developed and fine-tuned a **Convolutional Neural Network (CNN)** using popular frameworks like **TensorFlow** and **Keras** for image-based plant leaf count prediction[cite: 8, 70].
    • [cite_start]
  • Achieved a high classification accuracy of **92%** on the model[cite: 70].

Data and Model Robustness

    [cite_start]
  • To improve training dataset quality and boost model performance, I created a custom **image enhancement algorithm** using **histogram equalization**[cite: 19, 71].
    • [cite_start]
  • This preprocessing technique improved the model's overall **robustness and performance by 8%**[cite: 71].
    • [cite_start]
  • Automated and scaled data augmentation pipelines within Keras, resulting in a **20% reduction in total model training time**[cite: 72, 74].

Model Optimization

    [cite_start]
  • Applied rigorous model optimization techniques, including **K-fold cross-validation** and **grid search** for hyperparameter tuning, which enriched my machine learning expertise[cite: 20, 75].

Technologies Used

**Python**, **TensorFlow**, **Keras**, **Scikit-learn**, Deep Learning (CNN), Image Processing, Git.
