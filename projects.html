Projects

Educational Query Bot

Description: End-to-end NLP QA system that processes lecture videos, transcribes audio (ASR), builds semantic embeddings with BERT and retrieves context-aware answers using ALBERT. Designed for educational use-cases and scalable retrieval.

Paper: A Complete Chatbot based Architecture for answering user's Course-related queries in MOOC platforms

Tech: Python, ASR, BERT, ALBERT, Flask, Docker

EVs, GHG, and Indian Auto Sector

Description: Time-series forecasting to model the impact of EV adoption on greenhouse gas emissions. Built ARIMA and Prophet models, performed policy analysis and produced visualizations for stakeholders. Published on IEEE Xplore.

Paper: Predicting The Effect On Energy Sector And Greenhouse Gas (GHG) Emissions Due To The Electrification Of The Indian Automobile Sector

Tech: Python, Prophet, Scikit-learn, Pandas, Data viz

Image Super Resolution (SRCNN)

Goal: Recover a high-resolution image from a low-resolution input using SRCNN (Chao Dong et al., 2014).

Details: Implemented SRCNN in Keras, trained on example datasets, evaluated with PSNR, MSE and SSIM. Employed data augmentation and tuned learning schedules to improve reconstruction quality.

Tech: Keras, TensorFlow, Python, OpenCV

Life Expectancy Prediction

Description: Modelled life expectancy using WHO datasets. Compared linear regression, random forest and gradient boosting methods with cross-validation and feature engineering to select important indicators.

Tech: Python, Pandas, Scikit-learn

Optical Character Recognition (OCR) — Leaving Certificate Generator

Description: Automated generation of Leaving Certificate: system validates uploaded images/PDFs using OCR and image checks, then generates a formatted certificate when validation passes.

Tech: Python, Tesseract OCR, OpenCV
